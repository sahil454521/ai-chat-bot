
# AI Chat Bot

An intelligent and interactive chat bot application built using **TypeScript**, **CSS**, and **JavaScript**. This project demonstrates a powerful AI-driven conversational interface for various use cases, such as customer support, personal assistance, and more.

---

## Features

- **AI-Powered Conversations**: Leverages advanced AI algorithms to provide context-aware and human-like responses.
- **Customizable**: Easily adaptable to different domains and applications, allowing you to tailor the bot to your specific requirements.
- **Responsive Design**: Built with CSS to ensure a seamless user experience across devices, including desktops, tablets, and mobile phones.
- **Scalable Architecture**: Designed with scalability in mind to handle increasing user demands efficiently.

---

## Technologies Used

The project is developed using the following technologies:

- **TypeScript** (89.6%): The core programming language, offering type safety and robust development features.
- **CSS** (9.4%): Used for designing an aesthetically pleasing and responsive user interface.
- **JavaScript** (1%): Provides additional scripting and interactivity.

## Installation

Follow these steps to get the project up and running on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/sahil454521/ai-chat-bot.git
   cd ai-chat-bot
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm start
   ```

4. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```
